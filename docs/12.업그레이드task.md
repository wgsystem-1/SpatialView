# 업그레이드 Task 목록 (실행 계획)

`docs/12.향후발전계획.md`를 실제 실행 가능한 Task로 분해한 문서입니다.

---

## 1. 진행 원칙

- 벤치마킹 기반 목표 수립 → KPI 측정 → 개선 반복
- 사용성 개선(UX)과 성능 최적화를 병행
- 포맷 확장은 벡터 → 래스터 → 서비스/DB 순으로 단계 확대

## 3. KPI 측정 항목

- 로딩 시간(초)
- 선택 반응(ms)
- 하이라이트 표시(ms)
- 메모리 사용량 피크(MB)
- 저장 성공률(%)

---

## 4. 진행 상태 관리

- `ToDo → InProgress → Review → Done` 흐름으로 관리
- Task별 결과는 벤치마크 수치와 함께 기록
